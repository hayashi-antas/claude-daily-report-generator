# 日報ジェネレーター

「今日やったこと」を入力すると、上司に提出できる丁寧な日報文を自動生成するツールです。

## 特徴

- ルールベースで日報を生成（外部APIやLLM不要）
- 文末を自動的に「です/ます」調に変換
- 入力内容はブラウザに自動保存
- スマホ・PC両対応のレスポンシブデザイン

## 使い方

1. `index.html` をブラウザで開く
2. 「今日やったこと」を箇条書きで入力
3. 必要に応じて「学び・気づき」「明日の予定」を入力
4. 「日報を生成する」ボタンをクリック
5. 生成された日報をコピーして利用

## 技術スタック

- HTML5 / CSS3 / JavaScript (Vanilla)
- Tailwind CSS (CDN)
- ビルド不要・単一HTMLファイル構成
